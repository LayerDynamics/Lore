# Add Task

Quickly add a new task to your board with priority and due date.

## What You'll Be Asked

1. **Task title** (required)
2. **Which list?** (Upcoming, Today, Working on, etc.)
3. **Priority** (High, Medium, Low - optional)
4. **Due date** (optional)
5. **Time estimate** (optional)

The task will be created at the top of the selected list.
